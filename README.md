#castle-ella

- - -

View theme here: [Castle Ella](http://castle-ella.tumblr.com/)

A simple theme designed for Tumblr. 

###Theme Features

* 400px/500px Post
* Single Column Post
* 5 Customizable Links
* Toggle Reblog and Like Buttons
* Toggle Tags
* Toggle Footer
* Toggle Caption
* Custom Colors
* Custom Background/Post Images

###Other Features
* Sticky Navigation
* Fade in effect on page load
